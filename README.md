# Mini Fintech Dashboard

A MERN-based personal finance tracker built for the CiferX SDE Internship Assignment.

## Features

- Add transactions
- Filter transactions by category
- Summary cards
- Spending insights
- Pie chart visualization
- Simple login system
- Responsive fintech UI

## Tech Stack

- React + Vite
- Node.js + Express
- MongoDB Atlas
- Chart.js
- React Router

## Run Locally

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
